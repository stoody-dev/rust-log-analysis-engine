# 🚀 Parallel Rust Log Analysis Engine

High-performance CLI tool written in **Rust** for analyzing large log files using **streaming processing and parallel computation**.

Designed to efficiently process **millions of log entries** with minimal memory usage.

---

# ⚡ Quick Start
Clone and run in **3 commands**.

git clone https://github.com/stoody-dev/rust-log-analysis-engine.git

cd rust-log-analysis-engine

cargo run -- logs/sample.log

---

# ✨ Features

- ⚡ **Streaming log processing** (memory efficient)
- 🧵 **Parallel analysis** using Rayon
- 🔍 **Regex filtering** for log search
- 📊 **Log statistics generation**
- 💻 **CLI interface** using Clap
- 📈 **High performance (1M+ logs tested)**
